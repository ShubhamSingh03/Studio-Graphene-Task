## Studio Graphene Task

This repository contains the tasks for the Studio Graphene project.

## Responsive Screen Widths

Small screen: 376px to 1440px

Large screen: 1440px and above

These are the responsive widths provided in figma file.

## Getting started

> **Prerequisites:**
> The following steps require [NodeJS](https://nodejs.org/en/) to be installed on your system, so please install it beforehand if you haven't already.

To get started with project, you'll first need to install the dependencies with:

```
npm install
```

Then, to be able to run a development version of this project with:

```
npm run dev
```

After a few seconds, project should be accessible at the address
[http://localhost:1234/](http://localhost:1234/)
